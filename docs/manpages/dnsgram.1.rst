dnsgram
=======

Synopsis
--------

:program:`dnsgram` *INFILE*...

Description
-----------

:program:`dnsgram` takes one or more *INFILE*\ s in PCAP format and generates
statistics on 5 second segments allowing the study of intermittent
resolver issues.

Options
-------

None

See also
--------

pcap(3PCAP), tcpdump(8)
